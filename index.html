Initialization Options Dictionary

Pass these options to the second argument of the new VibeTabs() constructor:

activeTabClass

String Default: "VibeTabs--active"

The exact CSS class toggled onto the active <li> element wrapper container when selected.

remember

Boolean Default: false

When explicitly configured to true, tab tracking keys are pushed straight into the browser query history stack.

onChange

Function | null Default: null

Lifecycle tracking hook executed during navigation state shifts. Exposes an event object containing contextual elements: 

onChange: function(event) {
    console.log(event.tab);   // Target <a> anchor trigger element
    console.log(event.panel); // Coinciding HTML content wrapper panel
}

Public API Method Reference

.switch(input, updateURL = true)

Triggers a manual component navigation state switch. Accepts either a string targeting an element ID hash or a direct reference to a sub-child element trigger node.

.destroy()

Completely tears down the active module instance. Restores original inner HTML markup contexts, clears visibility restriction styles, and breaks internal memory storage references cleanly.
